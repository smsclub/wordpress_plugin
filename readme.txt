Send SMS all over the world from your website

== Description ==

Plugin from SmsClub company for sending SMS from your site.

1. Install the plugin to your Wordpress site.
2. Register on the website https://smsclub.mobi, go to the user profile and generate a token.
3. In the plugin settings, write down the username and token.
4. Now you can send SMS from the admin panel of your site.

== About ==

- The plugin implements interaction with the official API of the [SmsClub](https://smsclub.mobi) short message delivery service.

- This product provides SmsClub clients with the ability to easily integrate sending messages from their Wordpress site.
